Class PortfolioNamespace

The Portfolio namespace contains methods for getting data needed to build a portfolio.

Do not call this constructor directly. Instead, instantiate an Alchemy object with const alchemy = new Alchemy(config) and then access the portfolio namespace via alchemy.portfolio.
