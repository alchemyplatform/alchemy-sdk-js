    This method is useful when you want to refresh the metadata for a NFT that has been updated since the last time it was fetched. Note that the backend only allows one refresh per token every 15 minutes, globally for all users.

      The Alchemy SDK instance.

    • nft: BaseNft

      The NFT to refresh the metadata for.

    Returns Promise<boolean>

    This method is useful when you want to refresh the metadata for a NFT that has been updated since the last time it was fetched.
